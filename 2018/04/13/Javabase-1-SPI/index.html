本篇文章简要总结和分析下 Java 的 SPI 机制的原理。

1 概述

SPI，全称为 Sevice Provider Interface，是 JDK 内置的一种服务提供发现机制，用于框架扩展和代码替换，主要思想是将装配的控制权移到程序之外，在模块化设计中这个机制尤其重要，其核心思想就是解耦；

Android 中的 dagger 依赖注入框架的内部就有 SPI 的应用！
